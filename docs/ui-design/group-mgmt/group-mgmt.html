Create & name a group, invite people with a reusable 7-day link (share / copy / revoke), manage members (kick / leave), and delete a group — all on the chalkboard. One role-adaptive Group Settings page: admins get the full panel, members get the trimmed read-only view with “Leave”. Tapping the per-row gear button navigates directly to Group Settings (screen 2) — no intermediate action sheet.

AdminGroup Settings — the admin panel

Reached by tapping the per-row gear button on the Groups list (directly — no action sheet). Four sections: identity (+ rename), members (+ kick), invites (create inline — each row carries its own Share / Copy / Revoke), and a danger zone. Destructive acts get sheets: a simple confirm for kick/revoke, type-to-confirm for delete.
